# Shoehorn Homebrew Tap

[![OpenSSF Scorecard](https://api.scorecard.dev/projects/github.com/shoehorn-dev/homebrew-tap/badge)](https://scorecard.dev/viewer/?uri=github.com/shoehorn-dev/homebrew-tap)
[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

Homebrew tap for the [Shoehorn CLI](https://github.com/shoehorn-dev/shoehorn-cli).

## Usage

```bash
brew tap shoehorn-dev/tap
brew install shoehorn
```

## Update

```bash
brew upgrade shoehorn
```

Formula is updated automatically on each CLI release via GitHub Actions.
